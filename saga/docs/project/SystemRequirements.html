System requirementsÔÉÅ

You need a development environment with

  • Git

  • jq

  • the LTS release candidate of Node.js (current LTS release is version 18)

  • the current stable release of npm (version 8)

  • OpenSSL v3 (for the certificate generation helpers)

If you are using Windows, use the Windows Subsystem for Linux with Ubuntu 22.04 LTS.

Note

Windows is not included in the continuous integration tests, and hence if you encounter issues, you can vote in the nRF Asset Tracker project discussion.
