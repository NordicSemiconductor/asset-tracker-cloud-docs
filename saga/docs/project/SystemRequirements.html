System requirementsÂ¶

You need a development environment with Git, jq, the LTS release candidate of Node.js (current LTS release is version 14), and the current stable release of npm (version 7). Alternatively, you can install Node.js 15, which ships with npm 7.

If you are using Windows, use the Windows Subsystem for Linux with Ubuntu 20.04 LTS.

Note

Windows is not included in the continuous integration tests, and hence if you encounter issues, you can vote in the nRF Asset Tracker project discussion.
