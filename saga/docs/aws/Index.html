AWS cloud componentsÂ¶

The nRF Asset Tracker provides a reference implementation of a serverless backend for an IoT product using AWS. It is developed using AWS CDK in TypeScript.

The project also provides an easily understandable description of the available (and implemented) features, in a single folder called features.
