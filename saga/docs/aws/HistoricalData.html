Storage of historical dataÔÉÅ

The nRF Asset Tracker uses AWS Timestream to analyze the historical data. It offers developers a serverless and schemaless data storage optimized for IoT with the capability of querying using SQL.

AWS Timestream provides ease of maintenance, flexibility to change device messages, and cost-per-use pricing.
