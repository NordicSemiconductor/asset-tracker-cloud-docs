Storage of historical dataÂ¶

The nRF Asset Tracker uses AWS Timestream to analyze the historical data. This offers developers a serverless, schema-less data storage optimized for IoT with the capability of querying using SQL.

It provides ease of maintenance, flexibility to change device messages, and cost-per-use pricing.
