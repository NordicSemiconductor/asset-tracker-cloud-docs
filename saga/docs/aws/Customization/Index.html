Customization of the nRF Asset Tracker¶

This section walks you through the customization of the nRF Asset Tracker for your own product by showing how to visualize additional sensor data on the Cat Tracker web application.

Through the example, the following changes are made:

  1. Device Simulator UI is modified to generate the heart rate sensor data.

  2. Cat Tracker web application is modified to display the sensor data.
