Customization of the nRF Asset TrackerÂ¶

This section walks you through the customization of the nRF Asset Tracker for your own product. It shows how to visualize additional sensor data on the nRF Asset Tracker web application.

Through the example, the following changes are made:

  1. Device Simulator UI is modified to generate the heart rate sensor data.
  2. nRF Asset Tracker web application is modified to display the sensor data.
