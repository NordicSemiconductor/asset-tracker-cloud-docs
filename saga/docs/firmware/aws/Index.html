Firmware for the AWS cloud componentsÂ¶

The firmware of the nRF Asset Tracker is the nRF9160: Asset Tracker v2, which is a reference application developed using the nRF Connect SDK.

It is compatible with the nRF Asset Tracker for AWS.

The firmware GitHub repository provides an example for using the application as an out-of-tree development copy.
