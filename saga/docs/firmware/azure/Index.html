Firmware for the Azure cloud componentsÔÉÅ

The firmware of the nRF Asset Tracker is the nRF9160: Asset Tracker v2, which is a reference application developed using the nRF Connect SDK.

It is compatible with the nRF Asset Tracker for Azure.

The firmware GitHub repository provides an example for using the application as an out-of-tree development copy.

Note

The current firmware-over-the-air (FOTA) update mechanism implemented in nRF Asset Tracker for Azure is based on the Microsoft Azure tutorial for implementing a device firmware update process. In February 2021, Microsoft released a preview of Device Update for IoT Hub which will replace the current mechanism, as soon as it is generally available. See the Azure Device Update for IoT Hub support issue for details about the ongoing work to implement the new update mechanism.
