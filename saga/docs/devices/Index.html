Connect using a real deviceÔÉÅ

You can use any of the supported development kits to connect to your account.

To connect the device to your account, complete the following steps:

  1. Upgrade the modem firmware according to the instructions in the documentation:

    • nRF9160 DK modem upgrade

    • Thingy:91 modem upgrade

  2. Provision the certificate.

  3. Program the application firmware according to the instructions in the documentation:

    • nRF9160 DK application firmware update

    • Thingy:91 application firmware update
